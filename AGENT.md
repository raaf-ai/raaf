# RAAF Agent Guide (AGENT.md)

This document serves as the master index for AI agents working within the Ruby AI Agents Factory (RAAF) codebase. It connects all component-specific `CLAUDE.md` guides to provide a comprehensive map of the system.

## 🧠 System Context

RAAF is a Ruby implementation of AI Agents with **100% Python OpenAI Agents SDK feature parity**. It is organized as a mono-repo with specialized gems.

**Critical Architectural Pillars:**
1.  **Indifferent Access**: All contexts and results use `IndifferentHash`. `ctx[:key]` and `ctx["key"]` are identical.
2.  **Provider Neutrality**: Uses canonical token field names (`input_tokens`, `output_tokens`) and `ResponsesProvider` by default.
3.  **Python Parity**: Trace structure and agent behavior must align with Python SDK expectations.
